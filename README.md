# PrintSmart – Skip the Queue

## Overview
PrintSmart is a modern web application for hassle-free printing services. Users can upload files, customize options, pay, and track orders. Admins manage orders via a dashboard.

## Features
•⁠  ⁠*User Side*: Home, Login/Signup, Upload, Payment, Tracking.
•⁠  ⁠*Admin Panel*: Login, Dashboard with order management.
•⁠  ⁠*Tech Stack*: HTML/CSS/JS frontend, Flask backend.
•⁠  ⁠*UI*: Futuristic with gradients, animations, responsive design.

## Setup
1.⁠ ⁠Install Python 3.x.
2.⁠ ⁠Run ⁠ pip install -r requirements.txt ⁠.
3.⁠ ⁠Execute ⁠ python app.py ⁠.
4.⁠ ⁠Visit ⁠ http://127.0.0.1:5000/ ⁠ for user side, ⁠ /admin ⁠ for admin.

## Notes
•⁠  ⁠Payments are simulated.
•⁠  ⁠Files stored in ⁠ uploads/ ⁠, QR codes in ⁠ static/qr_codes/ ⁠.
•⁠  ⁠For production, add database, secure payments, and file handling.
